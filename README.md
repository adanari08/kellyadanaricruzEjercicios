# 🌐 Portafolio de Ejercicios de Programación Web

Este repositorio contiene una colección de ejercicios realizados como parte de la materia de **Programación Web**. El objetivo de estas actividades es practicar los conceptos fundamentales del desarrollo web utilizando **HTML**, **CSS** y **JavaScript**.

Cada ejercicio aborda un tema diferente, comenzando desde la estructura básica de una página web hasta la creación de formularios, tablas con estilos y aplicaciones sencillas con JavaScript.

---

##  Contenido del proyecto

| Ejercicio | Descripción |
|-----------|-------------|
| **Ejercicio 1** | Creación de una página "Hola Mundo" utilizando la estructura básica de HTML. |
| **Ejercicio 2** | Uso de etiquetas de texto, encabezados, párrafos, negritas y cursivas para dar formato al contenido. |
| **Ejercicio 3** | Implementación de enlaces internos y externos utilizando la etiqueta `<a>`. |
| **Ejercicio 4** | Inserción de imágenes mediante la etiqueta `<img>` y organización de archivos multimedia. |
| **Ejercicio 5** | Integración de texto, imágenes y enlaces en una pequeña página tipo blog personal. |
| **Ejercicio 6** | Uso de etiquetas semánticas como `<header>`, `<section>` y `<footer>` para estructurar una página web. |
| **Ejercicio 7** | Elaboración de listas ordenadas y desordenadas mediante una receta de cocina. |
| **Ejercicio 8** | Creación de un horario de clases utilizando tablas HTML y estilos CSS. |
| **Ejercicio 9** | Desarrollo de un formulario de contacto con un diseño moderno utilizando HTML y CSS. |
| **Ejercicio 10** | Desarrollo de un conversor de temperatura de Celsius a Fahrenheit utilizando HTML, CSS y JavaScript con validaciones y alertas mediante SweetAlert2. |

##  Cómo ejecutar el proyecto

1. Clona este repositorio:

```bash
git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
```

2. Ingresa a la carpeta del proyecto.

3. Abre el archivo **index.html** en cualquier navegador web moderno.

4. Desde el menú principal podrás acceder a cada uno de los ejercicios.
